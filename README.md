A Python/tkinter task management application built with the MVC (Model-View-Controller) pattern.

## Features
- Add tasks by typing and pressing **Enter** or clicking **Add**
- Mark tasks as done / undo with **Mark Done / Undo** or by double-clicking a task
- Delete selected tasks with **Delete**
- Live stats bar showing total, done, and remaining tasks

## Requirements
- Python 3.x (tkinter is included in the standard library)

## Running
```bash
python main.py
```
